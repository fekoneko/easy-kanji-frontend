# Easy Kanji
The website that might help you learn kanji easily

*Here you can find frontend part of the website. Made with Vite + React*

[*Follow this link for backend repository*](https://github.com/r4p1er/EasyKanjiServer)

## Feachures available on The 5th of May, 2023
- Popular Kanji List
- Select a kanji to learn it
- Learn mode (by meaning and by writing, you can select, flip and shuffle cards)
- Kanji search with authomatic romaji to kana conversion
- Registration and authorization with JWT tokens and role system (User and Admin roles)
- User can save (favorite) kanjis to his account
- User can edit his credentials
- User can leave a feedback on a site
- Admin have access to kanji editing page. Here they can edit, add or delete kanji
